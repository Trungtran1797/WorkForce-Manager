# WorkForceManager.Domain

- Entities (Employee, Department, Project, Task, Attendance, LeaveRequest, Notification, ...)
- Value Objects
- Enums (TaskStatus, LeaveType, Role, ...)
- Domain Interfaces (IRepository, IUnitOfWork contracts)
- Domain Exceptions

Không phụ thuộc vào bất kỳ layer nào khác.
