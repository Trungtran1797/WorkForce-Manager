# WorkForce Manager

Hệ thống quản lý công việc, nhân sự và tiến độ dự án dành cho doanh nghiệp xây dựng, kỹ thuật, thiết kế và phát triển phần mềm.

> Tài liệu định hướng đầy đủ cho AI agent (kiến trúc, modules, coding standards, roadmap) nằm tại [`CLAUDE.md`](./CLAUDE.md).

## Tech Stack

- **Frontend:** React, Vite, TypeScript, Tailwind CSS, Shadcn/UI
- **Backend:** ASP.NET Core 9, Entity Framework Core (Clean Architecture, CQRS + MediatR)
- **Database:** SQL Server
- **Auth:** JWT + Refresh Token + Role-Based Permission

## Project Structure

```
.
├── backend/   - ASP.NET Core 9 Clean Architecture solution
├── frontend/  - React + Vite + TypeScript + Tailwind + Shadcn/UI
└── docs/      - Architecture diagrams, ERD, API docs
```
