📘 API Endpoints chính Swagger UI ↗
MethodEndpointMô tả
POST/api/auth/loginĐăng nhập lấy JWT token
POST/api/auth/refreshLàm mới access token
POST/api/auth/logoutĐăng xuất + revoke token
GET/api/auth/meThông tin người dùng hiện tại
MethodEndpointMô tả
GET/api/employeesDanh sách nhân viên (phân trang)
GET/api/employees/{id}Chi tiết nhân viên
POST/api/employeesThêm nhân viên mới
PUT/api/employees/{id}Cập nhật thông tin
DEL/api/employees/{id}Xoá mềm nhân viên
GET/api/departmentsDanh sách phòng ban
MethodEndpointMô tả
GET/api/tasksDanh sách công việc
POST/api/tasksTạo công việc mới
PATCH/api/tasks/{id}/statusCập nhật trạng thái (Kanban)
GET/api/projectsDanh sách dự án
POST/api/projects/{id}/membersGán thành viên vào dự án
MethodEndpointMô tả
POST/api/attendances/check-inCheck-in đầu ngày
POST/api/attendances/check-outCheck-out cuối ngày
GET/api/attendancesLịch sử chấm công
GET/api/reports/attendanceBáo cáo chấm công
MethodEndpointMô tả
GET/api/leave-requestsDanh sách đơn nghỉ phép
POST/api/leave-requestsGửi đơn xin nghỉ
PUT/api/leave-requests/{id}/approveManager/HR duyệt đơn
PUT/api/leave-requests/{id}/rejectTừ chối đơn nghỉ

🚀 Cách khởi động dự án
1
Khởi động Backend API
dotnet run --project backend/src/WorkForceManager.WebApi
→ Swagger: localhost:5244/swagger
2
Khởi động Frontend
cd frontend && npm run dev
→ App: localhost:5174
3
Hoặc chạy với Docker
docker-compose up -d
→ Tất cả services khởi động tự động
4
Đăng nhập và kiểm tra
Dùng tài khoản demo bên trái để đăng nhập và kiểm tra từng module.
